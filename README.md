## Install
```sh
$ git clone git@github.com:albno273/meow.git
```
```sh
$ mkdir config
$ cd config
$ touch key.json
```

```json
{
    "test": {

        "cons_sec":      "xxx",
        "acc_token":     "xxx",
        "acc_token_sec": "xxx"
    }
}
```
```sh
npm install
node app.js
```
